# Book Api Client

## Общее описания

Фронтенд-клиент в котором реализованы:

* Таблица всех книг (пагинация - 10 элементов на страницу)
* Форма создания книги
* Форма редактирования книги
* Кнопки удаления, редактирования, добавления книги
* Кнопка таблицы книг
